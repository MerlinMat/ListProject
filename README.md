# ListProject
todo list application built with React and Vite.

## Features

- Add new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Responsive design
- Modern UI with icons
  Installation:

1. Clone the repository:https://github.com/MerlinMat/ListProject.git
2. npm install
3. npm run dev
4. Open browser at http://localhost:5173
